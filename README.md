# Burn Injury Segmentation Page


## Pre-requirements
- node v14.15.4
- npm  7.5.2

## Installation

```
npm install 

cd backend  && npm install
```


## Usage

1. Launch backend
```
cd backend && node app.js
```

2. Open the webpage

for now, you can double click the public/index.html

or use

```
npm start 
```

the webpage will start at `localhost:3000`
